텍사스 홀덤에서 **프리플랍 어그레서(공격자)**가 플랍에서도 베팅을 이어가는 것 — 이것이 컨티뉴에이션 벳, 줄여서 **C-bet**입니다. 홀덤에서 가장 많이 사용되고, 동시에 가장 많이 오용되는 기술이기도 합니다.

이 가이드에서는 C-bet의 원리부터 GTO 기반 사이즈 선택, 보드 텍스처별 전략까지 완벽하게 분석합니다.

---

## C-bet이란? 기본 개념

**컨티뉴에이션 벳(Continuation Bet)**은 프리플랍에서 마지막으로 레이즈한 플레이어가 플랍에서도 베팅을 이어가는 행동입니다.

### 왜 C-bet을 하는가?

1. **이니셔티브 유지** — 프리플랍 어그레서는 강한 핸드 범위를 나타내기 때문에 플랍에서도 베팅권이 있음
2. **폴드 에쿼티 획득** — 상대가 미스한 경우(약 67%) 폴드를 유도
3. **밸류 추출** — 강한 핸드에서 팟을 키움
4. **레인지 보호** — 블러프와 밸류를 섞어 상대가 읽기 어렵게 만듦

---

## C-bet 사이즈 기준 (2026 GTO 기반)

| 상황 | 권장 사이즈 | 이유 |
|------|------------|------|
| 드라이 보드 + 헤즈업 | 팟의 25~33% | 상대 폴드 유도에 충분 |
| 웻 보드 + 헤즈업 | 팟의 50~75% | 드로우 비용 높임 |
| 멀티웨이 (3인+) | 팟의 50~75% | 여러 상대 동시 압박 |
| 올인 전략 (스택 얕을 때) | 팟의 75~100% | SPR 낮을 때 팟 커밋 |

### 핵심 원칙
- **사이즈가 클수록 블러프 비율 낮춰야** (큰 베팅 = 더 강한 핸드처럼 보여야 설득력 있음)
- **작은 사이즈는 높은 빈도 가능** (33% 베팅은 낮은 가격으로 많이 사용 가능)

## 보드 텍스처별 C-bet 전략

### 🟢 드라이 보드 (C-bet 유리)

**예시: K♠ 7♦ 2♣ (레인보우)**

- 드로우 없음 → 상대 폴드 유도 쉬움
- 작은 사이즈(33%)로도 충분히 효과적
- **C-bet 빈도: 높음 (60~70%)**

```
적합한 핸드: 에이스 하이, 미드 포켓 페어, K 포함 모든 핸드
불필요한 경우: 보드에 연결된 강한 핸드 (슬로우플레이 고려)
```

### 🟡 웻 보드 (선택적 C-bet)

**예시: A♥ J♣ T♥ (투톤, 연결됨)**

- 드로우 많음 → 상대가 쉽게 콜 가능
- 큰 사이즈(66~75%)로 드로우 비용 높여야
- **C-bet 빈도: 중간 (40~55%)**

```
적합한 핸드: 탑 페어 이상, 플러시 드로우+오버카드 (세미블러프)
피해야 할 핸드: 에쿼티 없는 순수 블러프
```

### 🔴 멀티웨이 팟 (C-bet 주의)

**3명 이상이 플랍을 본 경우**

- 한 명은 히트했을 가능성 높음
- C-bet 효율 급격히 하락
- **C-bet 빈도: 낮음 (25~35%)**

```
멀티웨이에서 C-bet 해야 할 때: 탑 페어 이상 강한 핸드, 너트 드로우
멀티웨이에서 체크해야 할 때: 미드 페어, 약한 드로우, 블러프성 핸드
```

## C-bet 하지 말아야 할 상황

### 1. 레인지 디스어드밴티지
상대의 레인지가 보드에서 더 유리할 때는 체크가 올바릅니다.

**예시: CO vs BB, 보드 7♦ 6♠ 2♥**
- BB는 76s, 65s, 22 등 이 보드에서 강한 핸드를 많이 보유
- CO는 이 보드에서 강한 핸드가 상대적으로 적음
- → 체크가 더 균형 잡힌 선택

### 2. 매우 낮은 폴드 에쿼티
- 상대가 큰 드로우를 가졌을 때 (폴드 거의 안 함)
- 상대의 팟 오즈가 콜하기에 충분할 때
- 매우 웻한 보드에서 약한 핸드로 C-bet = 에쿼티 손실

### 3. 트랩핑 (슬로우플레이)
- 너트 핸드(셋, 풀하우스 등)를 가진 경우
- 상대가 베팅하도록 유도하기 위해 체크 선택
- 특히 웻 보드에서 너트를 가진 경우 효과적

## 포지션과 C-bet

### IP(인포지션) — C-bet 효율 높음

인포지션에서의 C-bet은 여러 이점이 있습니다:
- 상대가 체크 시 바로 추가 정보 없이 베팅 가능
- 팟 컨트롤 옵션 유지
- 턴/리버에서도 선택권 풍부

**IP C-bet 권장 빈도: 55~65%**

### OOP(아웃오브포지션) — 신중하게

아웃오브포지션에서 C-bet은 추가 약점이 있습니다:
- 상대의 레이즈에 불리한 위치에 놓임
- 턴에서 상대가 먼저 행동 후 정보 얻음

**OOP C-bet 권장 빈도: 40~50%**

## C-bet 빈도 최적화

### 너무 높은 C-bet 빈도의 위험

C-bet 빈도가 70% 이상이 되면:
- 상대가 당신의 C-bet에 가볍게 레이즈 가능
- 약한 핸드로 C-bet하다 스택 잃는 상황 발생
- 블러프 캐처로 쉽게 콜 당함

### GTO 기준 C-bet 빈도표

| 상황 | GTO C-bet 빈도 |
|------|---------------|
| 헤즈업 + 드라이 + IP | 60~70% |
| 헤즈업 + 드라이 + OOP | 45~55% |
| 헤즈업 + 웻 + IP | 45~55% |
| 헤즈업 + 웻 + OOP | 35~45% |
| 3웨이 | 25~35% |
| 4웨이 이상 | 15~25% |

## 실전 예시

### 예시 1: 드라이 보드 헤즈업

**상황:** UTG 레이즈, BB 콜. 보드 K♠ 8♦ 3♣ (레인보우)

UTG 레인지: AA~22, AK~A8, KQ~KT, 브로드웨이 등 강한 프리플랍 레이즈 레인지

**GTO 선택:** 팟의 33% C-bet, 빈도 65%

**이유:**
- K 하이 드라이 보드에서 UTG 레인지가 강함
- 작은 사이즈로 폴드 유도 + 밸류 추출 동시 달성
- BB가 이 보드에서 콜하기 어려운 핸드 많음

### 예시 2: 웻 보드 멀티웨이

**상황:** BTN 레이즈, SB 콜, BB 콜. 보드 9♥ 8♥ 6♦ (3웨이)

**GTO 선택:** 체크 권장 (또는 팟의 50% 이상 강한 C-bet)

**이유:**
- 3웨이에서 누군가 히트했을 가능성 높음
- 웻 보드라 드로우가 폴드하지 않음
- 중간 강도 핸드는 체크로 팟 컨트롤

## 고급 C-bet 전략: 블록 베팅과 프로브 베팅

### 블록 벳 (Block Bet)

OOP에서 상대의 큰 베팅을 막기 위해 먼저 작은 베팅(팟의 20~25%)을 선택하는 전략.

- 약한 핸드를 저렴하게 강 또는 쇼다운까지 가져가는 목적
- 상대가 레이즈하면 항상 폴드 계획을 세워야 함

### 프로브 벳 (Probe Bet)

플랍에서 IP 플레이어가 체크한 후, OOP 플레이어가 턴에서 베팅하는 전략.

- IP의 약점을 탐지(Probe)하는 개념
- IP가 플랍 C-bet을 자제한 경우에 기회 생성
