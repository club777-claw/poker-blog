## 🃏 기본 게임 구조 용어

| 용어 | 의미 |
|------|------|
| **홀카드** | 각 플레이어에게 딜되는 개인 카드 2장 |
| **커뮤니티 카드** | 모든 플레이어가 공유하는 보드 카드 |
| **플롭(Flop)** | 첫 번째로 공개되는 커뮤니티 카드 3장 |
| **턴(Turn)** | 플롭 이후 공개되는 4번째 카드 |
| **리버(River)** | 마지막으로 공개되는 5번째 카드 |
| **쇼다운** | 여러 플레이어가 남아 패를 공개하는 최종 대결 |
| **팟(Pot)** | 현재 테이블에 쌓인 총 금액 |

## 💰 베팅 용어

| 용어 | 의미 |
|------|------|
| **스몰 블라인드** | 딜러 왼쪽 첫 번째가 강제로 내는 소액 배팅 |
| **빅 블라인드** | 딜러 왼쪽 두 번째가 내는 기본 배팅 금액 |
| **콜(Call)** | 현재 배팅 금액에 맞춰 게임 참여 |
| **레이즈(Raise)** | 배팅 금액을 올림 |
| **폴드(Fold)** | 핸드를 버리고 게임 포기 |
| **체크(Check)** | 배팅하지 않고 기회를 다음으로 넘김 |
| **올인(All-in)** | 보유한 칩 전부를 배팅 |
| **3-벳(3-bet)** | 레이즈에 대한 재레이즈 |
| **4-벳(4-bet)** | 3-벳에 대한 또 한 번의 레이즈 |
| **오버벳** | 팟 금액 초과 배팅 |

## 🎯 핸드 강도 용어

| 용어 | 의미 |
|------|------|
| **프리미엄 핸드** | AA, KK, QQ, AK 등 최상위 시작 패 |
| **수티드(Suited)** | 같은 문양의 두 카드 (예: A♠K♠) |
| **오프수트(Offsuit)** | 다른 문양의 두 카드 |
| **커넥터** | 연속된 숫자의 카드 (예: 9-8) |
| **탑 페어** | 보드에서 가장 높은 숫자와 페어를 이룬 경우 |
| **오버페어** | 보드 모든 카드보다 높은 포켓 페어 |
| **세트** | 포켓 페어 + 보드 동일 숫자로 트리플 완성 |
| **넛(Nuts)** | 현재 보드에서 가능한 최강 핸드 |
| **드로우** | 스트레이트/플러시 완성을 기다리는 미완성 핸드 |

## 📐 전략/이론 용어

| 용어 | 의미 |
|------|------|
| **GTO** | Game Theory Optimal — 수학적 최적 균형 전략 |

| **밸류벳** | 강한 핸드로 상대에게 돈을 더 받아내는 배팅 |
| **블러프** | 약한 핸드로 상대를 폴드시키는 배팅 |
| **세미블러프** | 드로우를 가진 상태의 블러프 (이길 가능성 있음) |
| **팟 오즈** | 콜 비용 대비 팟 규모 비율 (수익성 판단) |
| **임플라이드 오즈** | 현재 팟 외 미래 추가 수익 고려한 오즈 |
| **핸드 레인지** | 특정 상황에서 플레이어가 가질 수 있는 모든 핸드 범위 |
| **밸런스** | 블러프와 밸류의 균형을 유지하는 플레이 |
| **체크레이즈** | 먼저 체크 후 상대 배팅에 레이즈하는 기술 |

## 자주 묻는 질문 (FAQ)

**Q. 포커에서 '블라인드'란 무엇인가요?**  
블라인드는 카드를 받기 전 강제로 배팅하는 금액입니다. 딜러 왼쪽 첫 번째 플레이어가 스몰 블라인드, 두 번째가 빅 블라인드를 냅니다. 게임을 진행시키는 기본 구조입니다.

**Q. GTO 전략이란 무엇인가요?**  
GTO(Game Theory Optimal)는 어떤 상대를 만나도 장기적으로 손해 보지 않는 수학적 최적 전략입니다. 상대가 완벽하게 플레이해도 착취당하지 않는 균형 잡힌 플레이 방식입니다.

**Q. 팟 오즈는 어떻게 계산하나요?**  
팟 오즈 = 콜 금액 ÷ (팟 총액 + 콜 금액). 예: 팟 1000원, 콜 500원이면 500 ÷ 1500 = 33%. 내 승률이 33% 이상이면 콜이 수학적으로 올바릅니다.
