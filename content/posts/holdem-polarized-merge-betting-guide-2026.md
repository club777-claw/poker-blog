## 홀덤 폴라라이즈드 vs 머지 베팅이란?

포커에서 레인지를 어떻게 구성하느냐는 승패를 가르는 핵심이다. **폴라라이즈드(Polarized)**와 **머지(Merge)**는 베팅 레인지 구성의 두 가지 기본 패러다임이다.

> **핵심 한 줄 요약**: 폴라라이즈드 = 최강+블러프만 크게 베팅 / 머지 = 넓은 밸류 레인지 작게 베팅

---

## 폴라라이즈드 vs 머지 핵심 비교표

| 항목 | 폴라라이즈드 | 머지 |
|------|------------|------|
| 레인지 구성 | 넛 핸드 + 블러프 (양극단) | 탑페어~투페어 (넓은 밸류) |
| 베팅 사이즈 | 75~150% 팟 (대형) | 33~50% 팟 (소형~중형) |
| 목적 | MDF 압박, 폴드 에쿼티 확보 | 얇은 밸류 추출, 팟 점진적 성장 |
| 적합 스트리트 | 리버 (주), 턴 일부 | 플랍·턴 (주) |
| 포지션 | IP 유리 | OOP에서도 활용 가능 |
| 상대 반응 | 콜/폴드 양분 | 콜 유도 |
| 미디엄 핸드 처리 | 체크 또는 폴드 | 베팅 레인지에 포함 |

## 폴라라이즈드 전략 심화

### 언제 폴라라이즈드를 사용하나?

1. **리버에서 드로우 완성/미완성 분기가 명확할 때** — 플러시 드로우 완성 리버에서 완성 핸드(넛)와 블러프 핸드만으로 베팅 레인지 구성
2. **드라이 보드 + IP (인포지션)** — 상대 레인지가 좁아진 상황에서 오버베팅으로 압박
3. **레인지 어드밴티지 극단적일 때** — 에이스 하이 보드에서 레이저가 에이스 조합 압도적 보유

### GTO 블러프 비율 계산 (폴라라이즈드)

```
베팅 사이즈 75% 팟:
  MDF = 베팅/(베팅+팟) = 75/(75+100) = 43%
  → 상대 43% 이상 콜해야 방어 완료
  → 베터 블러프 허용 비율 = 1 - MDF = 57% → 실제 밸류 대비 33% 블러프

베팅 사이즈 100% 팟:
  MDF = 100/200 = 50%
  → 블러프 허용 비율 = 25% (밸류 3 : 블러프 1)
```

### 폴라라이즈드 레인지 구성 예시

**리버 Kh Td 7s 2c Js (플러시 미완성)**

| 레인지 구분 | 핸드 예시 |
|-----------|---------|
| 넛 밸류 (베팅) | KK, TT, 77, KT, KJ |
| 블러프 (베팅) | 백도어 플러시드로우 미완성, 에이스 하이 |
| 미디엄 (체크) | 탑페어 약한 킥커, 미들페어 |

## 머지 전략 심화

### 언제 머지를 사용하나?

1. **플랍~턴 초중반 스트리트** — 드로우 가능성 많아 레인지 넓게 유지
2. **웻 보드 + OOP (아웃오브포지션)** — 작은 사이즈로 레인지 보호하며 밸류 추출
3. **상대가 공격적이지 않을 때** — 피시(Fish) 상대로 얇은 밸류 극대화

### 머지 레인지 구성 원칙

- **포함**: 탑페어~투페어 (강한 원페어~두 쌍)
- **포함**: 오버페어, 세트 (머지 사이즈로 얇게 시작)
- **제외**: 블러프 핸드 (머지 사이즈로 블러프 = 상대에게 좋은 오즈)
- **제외**: 너트 핸드 (머지보다 폴라라이즈드 사이즈로 더 많이 추출 가능)

### 보드별 머지 권장 사이즈

| 보드 텍스처 | 권장 머지 사이즈 | 이유 |
|-----------|--------------|------|
| 드라이 보드 (A72r) | 25~33% 팟 | 상대 폴드 유도 불필요, 얇은 콜 유도 |
| 반-웻 보드 (J96r) | 33~50% 팟 | 드로우 오즈 견제 + 밸류 추출 균형 |
| 웻 보드 (JTs8s) | 40~50% 팟 | 드로우에 나쁜 오즈 제공하며 밸류 |
| 멀티웨이 팟 | 33% 이하 | 여러 상대 레인지 고려, 얇게 |

## 포지션별 전략 가이드

| 포지션 | 폴라라이즈드 활용 | 머지 활용 |
|-------|--------------|---------|
| BTN (IP) | 리버 대형 베팅, 드라이 보드 오버베팅 | 플랍 1/3 팟으로 빈번하게 |
| CO (IP) | BTN과 유사, 조금 더 보수적 | 탑페어+ 머지 빈번 |
| BB (OOP) | 리버 한정, 넛 핸드만 | 플랍 소형 선베팅, 웻 보드 |
| SB (OOP) | 리버 제한적 사용 | 플랍 소형 머지 위주 |

## 폴라라이즈드 vs 머지 선택 기준 (실전 체크리스트)

```
✅ 폴라라이즈드 선택 시:
  □ 리버 스트리트인가?
  □ 드로우 완성/미완성으로 레인지 분기됐는가?
  □ 넛 어드밴티지가 압도적인가?
  □ 상대 레인지가 좁아 폴드 에쿼티 있는가?

✅ 머지 선택 시:
  □ 플랍 또는 턴 초반인가?
  □ 여러 핸드로 밸류를 추출하고 싶은가?
  □ OOP 포지션인가?
  □ 상대가 폴드보다 콜을 자주 하는 타입인가?
```

## 실전 예시

**예시 1 — 리버 폴라라이즈드 성공**
- 핸드: BTN Kh Kd / 상대: BB
- 보드: As Td 4c - 2h - 7s (드라이 런아웃)
- 리버 분석: 상대 레인지 = 탑페어(에이스) 중심. 우리 KK는 넛 블러프캐처 대상 아님
- 전략: 폴라라이즈드 100% 팟 → 상대 에이스 박킥커 폴드 유도 가능
- 결과: 상대 AT 폴드 (오버베팅 압박)

**예시 2 — 플랍 머지 베팅**
- 핸드: BB AhJd / 상대: BTN (레이저)
- 보드: Ac 7s 3d (드라이, OOP)
- 전략: 탑페어+탑킥커로 33% 팟 머지 베팅 → 상대의 포켓 페어, 세컨드페어 콜 유도
- 이유: 100% 팟으로 베팅 시 약한 핸드 폴드 → 얇은 머지로 더 많은 밸류 추출

## 자주 하는 실수

| 실수 | 문제 | 해결 |
|------|------|------|
| 리버에서 미디엄 핸드를 대형 베팅 | 상대에게 쉬운 폴드 유도 | 미디엄 핸드는 체크 또는 소형 |
| 블러프를 머지 사이즈로 베팅 | 상대에게 좋은 콜 오즈 제공 | 블러프는 폴라라이즈드 대형 or 포기 |
| 넛 핸드를 항상 머지 | 밸류 추출 기회 손실 | 넛은 폴라라이즈드로 사이즈 업 |
| 멀티웨이에서 폴라라이즈드 | 한 명이 반드시 콜 → 블러프 실패 | 멀티웨이는 넛만 대형 베팅 |

## 핵심 요약

- **폴라라이즈드** = 리버, 드라이 보드, IP, 넛+블러프 조합, 대형 사이즈
- **머지** = 플랍~턴, 웻 보드, OOP, 넓은 밸류 레인지, 소형 사이즈
- **미디엄 핸드** = 폴라라이즈드에서는 체크, 머지에서는 포함
- **블러프** = 항상 폴라라이즈드 사이즈 (소형 블러프 = GTO 위반)
