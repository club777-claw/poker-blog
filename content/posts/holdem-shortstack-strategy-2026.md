## 홀덤 숏스택이란?

**숏스택(Short Stack)**이란 테이블 평균 스택보다 칩이 현저히 적은 상태를 말한다.

일반적인 숏스택 기준:

| 스택 크기 | 분류 | 전략 방향 |
|----------|------|----------|
| 20BB 이상 | 일반 스택 | 표준 전략 사용 가능 |
| 10~20BB | 숏스택 | 제한된 포스트플랍, 올인 혼용 |
| 10BB 이하 | 극단 숏스택 | 올인(JAM) 전략 전환 필수 |
| 5BB 이하 | 푸시/폴드 모드 | 핸드 강도와 포지션만 고려 |

스택이 적을수록 **복잡한 포스트플랍 전략 대신 프리플랍 올인 결정**이 핵심이 된다.

---

## 숏스택의 근본 문제

스택이 줄어들면 다음 문제가 발생한다:

1. **팟 오즈 계산이 달라진다** — 콜해야 할 핸드 범위가 넓어짐
2. **블러핑 효과가 사라진다** — 포스트플랍 배팅 공간이 없음
3. **레버리지 상실** — 스택 크기로 상대를 압박할 수 없음
4. **ICM 압박(토너먼트)** — 탈락 위험이 곧 현금 손실로 연결

따라서 숏스택 전략의 핵심은 **"언제 올인할 것인가"를 미리 결정**하는 것이다.

## 스택별 핵심 전략

### 10~20BB — 전환 구간

이 구간에서는 오픈레이즈(2~2.5x)와 올인을 혼용한다.

**포지션별 오픈 레인지 (15BB 기준)**

| 포지션 | 오픈 핸드 | 전략 |
|--------|----------|------|
| BTN | 45~50% | 폭넓게 오픈, 팟 컨트롤 |
| CO | 35~40% | 중간 강도 이상 |
| HJ | 25~30% | 타이트하게 |
| UTG | 15~20% | 강한 핸드만 |

**핵심 원칙:**
- 오픈 후 3벳 당하면 → **폴드 또는 올인** (콜은 EV 손실)
- 포스트플랍으로 가면 스택이 너무 얕아져 옵션이 없어짐

### 10BB 이하 — JAM(올인) 모드

10BB 이하에서는 **올인 or 폴드**만 존재한다. 오픈레이즈는 없다.

**푸시/폴드 핸드 차트 (10BB 기준)**

| 포지션 | 올인 가능 핸드 |
|--------|--------------|
| BTN | A2o+, K9o+, Q9o+, J9o+, T9o, 22+ |
| CO | A3o+, KTo+, QTo+, JTo, 55+ |
| HJ | A7o+, KJo+, QJo, 77+ |
| UTG | ATo+, AJs+, KQo+, 99+ |
| SB | A2s+, A5o+, K8s+, KTo+, Q9s+, 66+ |

**5BB 이하 — 어디서든 강한 핸드는 올인**
- AXs, KQo, 88+ → 포지션 무관 올인
- A8o+, KJs+ → 후반 포지션 올인

## 콜 vs 올인 결정 기준

상대가 오픈레이즈 했을 때 숏스택으로 콜 vs 올인을 결정하는 기준:

### 팟 오즈 계산

> **필요 콜 승률(%) = 콜 금액 ÷ (현재 팟 + 콜 금액) × 100**

예시:
- 상대 오픈 2.5BB, 내 스택 12BB
- 콜 시 잔여 스택: 9.5BB → 포스트플랍 공간 협소
- **올인(12BB)** 하면 상대 콜 강요 → 약한 핸드 폴드 유도 가능

**콜보다 올인이 나은 경우:**
- 스택의 50% 이상 콜해야 할 때
- 포스트플랍 공간이 사실상 없을 때
- 핸드가 블러프 캐처 이상의 가치일 때

## 토너먼트 vs 캐쉬게임 숏스택 전략 차이

### 토너먼트 (ICM 적용)

**ICM(Independent Chip Model)** — 칩 가치가 선형적이지 않음. 탈락하면 상금 0.

| 상황 | ICM 조정 |
|------|---------|
| 버블 근처 | 더 타이트하게 (탈락=큰 손실) |
| 숏스택 우선 처리 | 롱스택에 불리 (탈락 압박) |
| 파이널 테이블 진입 전 | 극단적 타이트 가능 |

**토너먼트 숏스택 핵심:**
- 상금 점프 구간 직전 → 평균보다 더 타이트
- 버블에서 롱스택 = 올인 공격 적극적 가능
- 스택 20BB 미만이면 매 핸드 올인 여부 즉시 판단

### 캐쉬게임

- 탈락 = 리바이(rebuying) 가능 → ICM 없음
- 순수 EV 기반 결정
- 리바이 타이밍: 대부분 100BB 이하면 리바이 권장

**캐쉬게임 숏스택 조언:**
- 고의 숏스택(short-stacking) 전략도 존재 — 단순화된 게임 선호할 때
- 단, 장기적으로 풀스택 대비 EV 불리 → 가능하면 리바이

## 숏스택 상대 공략법

내가 롱스택일 때 숏스택 상대를 상대하는 법:

### 아이솔레이션(Isolation) 전략

숏스택 오픈에 리레이즈 → 헤즈업 강제 → 상대 레인지 좁힘

**아이솔레이션 레이즈 크기:**
- 숏스택 올인에 콜: 팟 오즈로 판단
- 숏스택 오픈에 3벳: 4~5x (나머지 참가자 배제 목적)

### 숏스택 콜 레인지 계산

상대 10BB 올인에 SB에서 콜할 때:

```
팟 오즈 = 콜 금액 / (팟 + 콜 금액)
= 9.5BB / (10 + 0.5 + 9.5)BB = 9.5/20 ≈ 47.5%
→ 핸드 승률 47.5% 이상이면 콜 OK
```

일반적으로 A8o, KTo, 55+ 이상이면 콜 가능.

## 실전 예시

### 예시 1: 토너먼트 버블 숏스택

**상황:** 20명 남음, 버블(18명 입상). 내 스택 12BB, BTN.
- 폴드 → 폴드 → CO폴드 → 내가 BTN, K9o
- **판단:** 버블이지만 K9o는 BTN 12BB 올인 레인지. 올인 → 블라인드 폴드 → +3BB 획득
- **교훈:** 버블에서도 숏스택이면 스틸 올인이 필수. 폴드하면 스택이 더 줄어든다

### 예시 2: 캐쉬게임 8BB 숏스택

**상황:** 100BB 캐쉬게임, 나 8BB. UTG+1 A7s 손에 쥠.
- **판단:** A7s는 UTG+1 8BB에서 올인 레인지. → 올인 → BTN 콜(AKo) → 아웃드로우 실패
- **결론:** 콜이 맞는 판단이었다. 결과와 과정을 분리해서 판단해야 한다
- **교훈:** 숏스택에서 EV+ 결정을 반복하면 장기적으로 이긴다

## 숏스택 전략 요약

| 스택 | 핵심 행동 |
|------|----------|
| 20BB+ | 표준 전략, 포지션 활용 |
| 15~20BB | 오픈레이즈 + 올인 혼용 |
| 10~15BB | 올인 비중 높이기 |
| 5~10BB | 올인/폴드 모드 |
| 5BB 이하 | 어디서든 AX+, 66+ 올인 |

**핵심 원칙:**
1. 스택 10BB 이하 → 오픈레이즈 금지, 올인만
2. 포지션이 좋을수록 올인 레인지 확장
3. 토너먼트 버블 → ICM 고려해 약간 더 타이트
4. 콜보다 올인이 EV 높을 때가 많다
5. 롱스택으로 숏스택 만나면 아이솔레이션 활용
