## 포커 핸드 순위란?

포커 핸드 순위란 카드 5장 조합의 강약을 정한 체계입니다. 텍사스 홀덤, 오마하 등 대부분의 포커 게임에 동일하게 적용됩니다.

쇼다운(카드 공개) 시 더 높은 순위의 핸드를 가진 플레이어가 팟을 가져갑니다.

## 핸드별 실전 예시

### 1위 — 로얄 플러시
```
♠A  ♠K  ♠Q  ♠J  ♠10
```
같은 무늬(♠)의 A, K, Q, J, 10. 게임에서 절대 지지 않는 최강 핸드.

### 2위 — 스트레이트 플러시
```
♥9  ♥8  ♥7  ♥6  ♥5
```
같은 무늬로 연속된 5장. 로얄 플러시가 아닌 경우.

### 3위 — 포카드 (Four of a Kind)
```
K♠  K♥  K♦  K♣  3♠
```
같은 숫자 4장. 나머지 1장(킥커)은 동점 시 결정에 사용.

### 4위 — 풀하우스
```
Q♠  Q♥  Q♦  8♣  8♠
```
같은 숫자 3장 + 같은 숫자 2장 조합.

### 5위 — 플러시
```
♦A  ♦J  ♦8  ♦6  ♦3
```
같은 무늬 5장. 숫자가 연속되지 않아도 됨.

### 6위 — 스트레이트
```
10  9  8  7  6  (무늬 무관)
```
연속된 숫자 5장. A는 최고(A-K-Q-J-10) 또는 최저(A-2-3-4-5) 모두 가능.

### 7위 — 쓰리 오브 어 카인드 (트리플)
```
7♠  7♥  7♦  A  K
```
같은 숫자 3장.

### 8위 — 투페어
```
A♠  A♥  K♠  K♥  Q
```
같은 숫자 쌍 2세트.

### 9위 — 원페어
```
J♠  J♥  A  K  Q
```
같은 숫자 쌍 1세트.

### 10위 — 하이카드
```
A  K  9  7  3  (무늬 다름, 연속 아님)
```
아무 조합도 없는 상태. 가장 높은 카드로 비교.

## 포커 핸드 순위 외우는 비법 5단계

**1단계 — 최상위 2개**
로열 플러시 > 스트레이트 플러시 (같은 무늬가 핵심)

**2단계 — 세트류**
포카드 > 풀하우스 (4장이 3장+2장보다 강함)

**3단계 — 무늬/연속**
플러시 > 스트레이트 ("플"이 "스"보다 앞)

**4단계 — 페어류**
쓰리 오브 어 카인드 > 투페어 > 원페어

**5단계 — 최하위**
하이카드 = 아무 조합 없음

## 동점 처리 규칙

같은 순위의 핸드끼리 맞붙을 때:
- **원페어/투페어/트리플:** 더 높은 숫자의 페어가 승리
- **풀하우스:** 3장 부분의 숫자가 더 높은 쪽 승리
- **플러시/하이카드:** 가장 높은 카드부터 순서대로 비교
- **완전 동점:** 팟 스플릿 (나눠 가짐)
