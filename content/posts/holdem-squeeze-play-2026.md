## 홀덤 스퀴즈 플레이란?

**스퀴즈(Squeeze)**란 오픈레이저 + 1명 이상의 콜러가 있는 상황에서 **3벳으로 팟을 크게 올리는 전략**입니다.

"스퀴즈"라는 이름은 오픈레이저와 콜러들 사이에 끼어 있는 상황에서 **그들을 양쪽에서 압박(squeeze)**하는 데서 유래했습니다.

### 스퀴즈의 핵심 원리

| 요소 | 설명 |
|------|------|
| **폴드 에쿼티** | 콜러들이 끼면 오픈레이저도 콜드콜이 부담 → 폴드 확률 상승 |
| **팟 사이즈** | 콜러가 많을수록 팟이 커져 있어 스퀴즈 가치 상승 |
| **포지션 이점** | SB/BB에서 스퀴즈 후 헤즈업 → 상대 레인지가 압축됨 |

---

## 스퀴즈 실전 조건

### 최적 스퀴즈 상황

**✅ 스퀴즈 적합:**
- BTN/CO 넓은 오픈 + 1~2명 콜
- SB/BB/HJ 위치에서 스퀴즈
- 콜러가 수동적(패시브)인 경우

**❌ 스퀴즈 회피:**
- UTG 타이트 오픈 (레인지 강함)
- 콜러가 공격적 → 4벳 위험
- OOP에서 콜러가 3명 이상

## 포지션별 스퀴즈 빈도 (GTO 기준)

| 스퀴즈 위치 | 오픈 위치 | 콜러 수 | 스퀴즈 빈도 |
|------------|---------|--------|------------|
| BB | BTN | 1명 | 18~22% |
| BB | CO | 1명 | 14~18% |
| BB | HJ | 1명 | 10~14% |
| SB | BTN | 1명 | 20~25% |
| CO | HJ | 1명 | 8~12% |
| BB | BTN | 2명 | 25~30% |
| BB | CO | 2명 | 20~25% |

*콜러가 늘수록 스퀴즈 빈도 상승 (팟 규모 증가 + 폴드 에쿼티 증가)*

## 스퀴즈 핸드 구성

### 밸류 스퀴즈 핸드

| 핸드 | 특징 |
|------|------|
| AA, KK | 무조건 스퀴즈 (최강 핸드) |
| QQ, JJ | 대부분 스퀴즈 (상황에 따라 플랫도 가능) |
| AKs, AKo | 스퀴즈 우선 |
| AQs | 스퀴즈 or 플랫 (포지션/상대 따라) |

### 블러프 스퀴즈 핸드

| 핸드 | 이유 |
|------|------|
| A2s~A5s | 에이스 블락커 + 너트 플러시 가능성 |
| KQs | 킹/퀸 블락커 + 플랍 이후 연결성 |
| 67s, 78s, 89s | 수티드 커넥터 → 역전 가능성 |
| KJs | 블락커 효과 + 플랍 후 플레이 가능 |

**블러프 스퀴즈 원칙:** 플랍에서 계속 싸울 수 있는 형태의 핸드 선택

## 스퀴즈 사이즈 계산법

### 기본 공식

```
스퀴즈 사이즈 = 오픈 사이즈 × 3~4 + 콜러 수 × BB
```

**실전 예시:**
- 오픈 3BB + 콜러 1명: 3 × 3.5 + 1 = **약 11~12BB**
- 오픈 3BB + 콜러 2명: 3 × 3.5 + 2 = **약 13~14BB**
- 오픈 2.5BB + 콜러 1명: 2.5 × 3.5 + 1 = **약 10BB**

### 스택별 조정

| 스택 (BB) | 스퀴즈 사이즈 | 비고 |
|-----------|-------------|------|
| 100BB+ | 공식 그대로 | 표준 |
| 50~100BB | 약간 줄임 | SPR 고려 |
| 30~50BB | 올인 스퀴즈 고려 | 폴드 에쿼티 극대화 |
| 30BB 이하 | 올인 또는 폴드 | 스퀴즈 범위 좁힘 |

## 스퀴즈 후 플랍 전략

### 헤즈업으로 갔을 때

**레인지 어드밴티지 활용:**
- 스퀴즈 레인지는 상대보다 압축됨 → 강한 레인지
- 건조한 보드(A72r, K83r): 높은 빈도 C-bet (65~75%)
- 젖은 보드(JT9ss, 678ss): 선택적 C-bet, 강한 핸드 집중

### 멀티웨이로 갔을 때

| 핸드 강도 | 전략 |
|---------|------|
| 탑 페어 이상 | 베팅 (작은 사이즈) |
| 중간 핸드 | 체크 (팟 컨트롤) |
| 블러프 핸드 | 대부분 체크 |

## 실전 예시

### 예시 1: BB 스퀴즈 성공 케이스

```
상황: 100BB 스택, 9명 테이블
CO: 3BB 오픈
BTN: 콜
SB: 폴드
BB (나): ATs 보유

→ 스퀴즈 12BB
→ CO, BTN 모두 폴드
→ 팟 7.5BB 수집
```

ATs는 블락커(에이스) + 플랍에서 플레이 가능 → 이상적인 블러프 스퀴즈 핸드

### 예시 2: 밸류 스퀴즈

```
상황: 100BB 스택
HJ: 3BB 오픈
CO: 콜
BTN: 콜
BB (나): KK 보유

→ 스퀴즈 14BB (오픈 3 × 3.5 + 콜러 2)
→ HJ 폴드, CO 폴드, BTN 콜
→ 헤즈업 플랍 진행
→ K♥7♦2♣ 보드: 전속력 벳
```

## 스퀴즈 대응법 (상대가 스퀴즈할 때)

| 내 포지션 | 핸드 | 대응 |
|---------|------|------|
| 오픈레이저 | AA, KK | 4벳 |
| 오픈레이저 | QQ, JJ | 4벳 or 콜 |
| 오픈레이저 | AKs | 4벳 |
| 오픈레이저 | TT 이하 | 대부분 폴드 |
| 콜러 | 수티드 커넥터 | 폴드 (팟 오즈 부족) |
| 콜러 | 포켓 페어 7~9 | 폴드 |

## 핵심 체크리스트

- [ ] 상대 오픈 위치와 레인지 확인 (UTG 타이트 → 스퀴즈 조심)
- [ ] 콜러 수 확인 → 많을수록 스퀴즈 가치 ↑
- [ ] 밸류/블러프 비율 유지 (50:50 or 밸류 약간 더)
- [ ] 블러프 스퀴즈는 블락커 + 역전 가능성 있는 핸드
- [ ] 스퀴즈 사이즈: 오픈×3.5 + 콜러×1BB
- [ ] 4벳 대응 계획 미리 세우기
