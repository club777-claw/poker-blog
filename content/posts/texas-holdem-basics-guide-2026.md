  - question: "텍사스 홀덤이란 무엇인가요?"
    answer: "텍사스 홀덤(Texas Hold'em)은 각 플레이어에게 비공개 카드 2장을 나눠준 뒤, 공개 카드 5장(커뮤니티 카드)과 조합해 최강의 5장 패를 만드는 포커 게임입니다. 세계에서 가장 많이 즐기는 포커 변형으로, WSOP 메인 이벤트에서도 사용됩니다."
  - question: "홀덤에서 블라인드란 무엇인가요?"
    answer: "블라인드는 카드를 받기 전에 강제로 베팅하는 금액입니다. 딜러 왼쪽 플레이어가 스몰 블라인드, 그 다음이 빅 블라인드를 냅니다. 빅 블라인드는 스몰 블라인드의 2배가 일반적입니다."
  - question: "홀덤 한 판은 어떻게 진행되나요?"
    answer: "프리플랍(홀 카드 2장 배분 후 베팅) → 플랍(커뮤니티 카드 3장 공개 후 베팅) → 턴(4번째 카드 공개 후 베팅) → 리버(5번째 카드 공개 후 베팅) → 쇼다운(패 비교) 순서로 진행됩니다."
  - question: "콜, 레이즈, 폴드는 무슨 뜻인가요?"
    answer: "콜(Call)은 현재 베팅금액에 맞춰 따라가기, 레이즈(Raise)는 베팅금액을 올리기, 폴드(Fold)는 이번 판을 포기하고 카드를 버리는 행동입니다. 체크(Check)는 자신 차례에 추가 베팅 없이 넘기는 것입니다."
  - question: "텍사스 홀덤을 처음 배울 때 가장 중요한 것은?"
    answer: "핸드 순위(족보)를 외우는 것이 첫 번째입니다. 로얄 플러시가 최강이고 하이카드가 최약입니다. 두 번째는 포지션 — 딜러 버튼에 가까울수록 유리합니다. 세 번째는 타이트하게 시작 핸드를 선택하는 것입니다."

텍사스 홀덤은 홀 카드 2장 + 커뮤니티 카드 5장 중 최강 5장 조합으로 승부하는 포커 게임입니다. 전 세계 카지노·온라인 플랫폼에서 가장 많이 플레이되는 포커 변형으로, WSOP 메인 이벤트의 공식 종목이기도 합니다.

---

## 한 판 진행 순서

| 단계 | 이름 | 내용 |
|------|------|------|
| 1 | **프리플랍** | 홀 카드 2장 배분 → 첫 베팅 |
| 2 | **플랍** | 커뮤니티 카드 3장 공개 → 베팅 |
| 3 | **턴** | 4번째 카드 공개 → 베팅 |
| 4 | **리버** | 5번째 카드 공개 → 최종 베팅 |
| 5 | **쇼다운** | 남은 플레이어 패 비교 → 승자 결정 |

> **핵심:** 각 단계마다 콜/레이즈/폴드/체크 중 하나를 선택. 가장 강한 5장 조합을 만든 플레이어가 팟을 가져갑니다.

## 필수 용어 정리

| 용어 | 뜻 |
|------|-----|
| **딜러 버튼(D)** | 이번 판의 기준점. 게임마다 시계 방향으로 이동 |
| **스몰 블라인드(SB)** | 딜러 왼쪽 첫 번째, 강제 소액 베팅 |
| **빅 블라인드(BB)** | SB 다음 자리, SB의 2배 강제 베팅 |
| **콜(Call)** | 현재 베팅에 맞춰 따라가기 |
| **레이즈(Raise)** | 베팅금액 올리기 |
| **폴드(Fold)** | 포기, 카드 반납 |
| **체크(Check)** | 추가 베팅 없이 넘기기 |
| **올인(All-in)** | 보유 칩 전부 베팅 |
| **팟(Pot)** | 이번 판에 쌓인 총 베팅금액 |
| **쇼다운(Showdown)** | 마지막 베팅 후 패를 공개해 승자 결정 |

## 초보자가 꼭 알아야 할 3가지 원칙

### 원칙 1: 족보(핸드 순위) 암기

강한 순서:
**로얄 플러시 → 스트레이트 플러시 → 포카드 → 풀하우스 → 플러시 → 스트레이트 → 트리플 → 투페어 → 원페어 → 하이카드**

처음에는 이 10개 순서만 외우면 됩니다.

### 원칙 2: 포지션 이해

늦게 행동할수록 정보가 많아 유리합니다.
- **딜러 버튼(BTN)** = 최고 포지션 (마지막으로 행동)
- **얼리 포지션(UTG)** = 최악 포지션 (가장 먼저 행동)

포지션이 좋을수록 더 많은 핸드를 플레이할 수 있습니다.

### 원칙 3: 타이트하게 시작

처음에는 강한 핸드만 참여하세요. 폴드는 손해가 아닙니다.

**추천 입문 핸드:** AA, KK, QQ, JJ, AK(수티드), AQ(수티드)
나머지는 과감히 폴드. 이것만 지켜도 손실이 크게 줄어듭니다.
